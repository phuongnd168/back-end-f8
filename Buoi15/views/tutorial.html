Welcome to Todo App!

    That some thing you must know:
  • This todo app does not require network to use!
  • This todo app will NOT save immediately when you make changes!
  • This todo app will ONLY save when you leave the page!
  • This todo app uses localStorage to store data(When leaving the page)!
  • This todo app uses toastify to make pop-up notifications!

So, how can i using this?

Step 1: Create a new todo

By typing in the input, submit. You can add new task very easy

If you add a "empty" task, don't worry, you can Update or Delete it!

Step 2: Update a todo

If you want to update a todo, just Double click on the content of todo you want to update.

By the "empty" task, if you change the content, you can see the id was changed!

So, how about done task? Make it done! or Delete it!

Step 3: Make it done!

If you want to change a status of todo, just One click on the status of todo you want to update.

Or, you can Delete it!

Step 4: Destroy it!

If you don't want to change a status of todo, just Double click on the ID of todo you want to delete.

Or, you can Change status!

For now, you can Create new task!

Let do it!
